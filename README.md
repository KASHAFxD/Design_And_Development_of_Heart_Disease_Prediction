# Heart Disease_prediction
####  Project Setup
Requirements:<br>
- 8 GB RAM<br>
- Code Editor VSCode / Pycharm<br>
- Git CLI<br>
- Docker Desktop<br>
- Web Browser Chrome / Edge<br>
- python 3.12.4<br>
  
#### Clone the Project

git@https://github.com/KASHAFxD/heart_disease](https://github.com/KASHAFxD/Design_And_Development_of_Heart_Disease_Prediction

http://localhost:8000/
